# 🛡️ Búnker IA (Qwen-Coder)

Sistema privado de Inteligencia Artificial para programar y chatear, optimizado para máquinas modestas.

## 🚀 Instalación Instantánea
Clona el repositorio y ejecuta el script:
\`\`\`bash
git clone https://github.com/bagpotato/bunker-ia.git
cd bunker-ia
chmod +x instalar.sh
./instalar.sh
\`\`\`

## 🛠️ Acceso
- **Local:** [http://localhost:3000](http://localhost:3000)
- **Remoto:** Usa la IP de **Tailscale** en el puerto \`:3000\`.

## 🧠 Hardware Recomendado
- Modelo por defecto: \`qwen2.5-coder:1.5b\`.
